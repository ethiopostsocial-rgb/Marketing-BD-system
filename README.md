# Ethiopost M&BD Working Platform

Internal platform for Ethiopost Marketing & Business Development team.

## Stack
- React 19 + TypeScript + Vite
- TanStack Router + TanStack Query
- Zustand (state) + Tailwind CSS v4
- Vercel (hosting + serverless API)
- GitHub (live database via API)
